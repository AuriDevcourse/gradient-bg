# Gradient Studio

An interactive gradient background generator with a beautiful admin panel for customizing animated morphing gradients.

## Tech Stack

### Core
- **Framework**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS

### UI & UX
- **Components**: shadcn/ui inspired design system
- **Animations**: Framer Motion
- **Icons**: Lucide React

### Backend (Ready for integration)
- **Database & Real-time**: Supabase (PostgreSQL)

## Features

- **Live Preview**: Real-time gradient customization
- **Interactive Controls**: Adjust colors, size, blur, and blend modes
- **Code Export**: Generate ready-to-use HTML, CSS, and JavaScript
- **Dark Theme**: Beautiful shadcn/ui inspired interface
- **Smooth Animations**: Powered by Framer Motion
- **Multiple Blend Modes**: Hard light, overlay, screen, multiply, and more

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Usage

1. Open the app in your browser
2. Use the left sidebar to customize your gradient
3. Adjust colors, size, blur intensity, and blend modes
4. Click "Export Code" to get the implementation code
5. Copy and paste into your project
